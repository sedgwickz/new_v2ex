## Features

- Blazing fast 飞快

- Global SSR 全局服务端渲染

- Responsive Layout 响应式布局，支持桌面和移动设备显示，（IE<=11未测试，可能存在部分布局问题）

- Multiple Themes 多主题，支持亮色和暗黑主题
